Agent SSE Flow

Free, unlimited React component for visualizing Agent execution traces. LangSmith alternative.

✅ 100% Free 🔒 Local & Private ⚡ 10K+ Events 📦 5-min Setup

🆓 Unlimited & Free

No trace limits, no monthly fees. Unlike LangSmith's 5000 traces/month limit.

🔒 Data Privacy

All data stays local. Your agent traces never leave your machine.

⚡ High Performance

Virtual scrolling handles 10,000+ events without breaking a sweat.

🎨 Beautiful UI

Dark/Light themes, timeline view, markdown support out of the box.

🎥 Live Demo

Quick Start

# Install
npm install agent-sse-flow

# Use
import { AgentFlow } from 'agent-sse-flow'
import 'agent-sse-flow/style.css'

function App() {
  return (
    <AgentFlow 
      url="http://localhost:8080/agent/stream"
      theme="dark"
    />
  )
}
